Unboxing the CSS Box Model

When laying out a page, the browser represents each element as a box with different areas, thus “the css box model”. These areas determine the space surrounding each element and how that space interacts with other elements in your page. Every HTML element has four areas: content, padding, border and margin. The image below illustrates the areas in the css box model:



Let's have a look at each area of the model:

    Content: The content area is the element itself (text, image, video player). Its dimensions are defined by the element's width and height. These dimensions are given by default but they can be customized with the widths and high properties.

     Padding: An element's padding area is the space between its content and its border. The CSS padding properties are used to generate space around an element's content, inside of any defined borders. Padding values are set using lengths, percentages, or the inherited keyword. Padding cannot accept negative values. The initial, or default, value for all padding properties is 0.

    The values can be set by the padding-top, padding-right, padding-left and padding-bottom properties or by the shorthand padding property. When using the shorthand padding property you can specify one, two, three or four values.

      - One value: sets the same padding in all 4 directions (top, right, bottom and left)
       - Two values: the first value sets the top and bottom padding, and the second value sets the right and left.
       - Three values: first one for top, second ones for right and left, and the third one for bottom.
       - Four values: the values represent the top, right, left and bottom padding.

    Padding can be used to make an element grow or shrink. Here is an example of how padding affects the content's size:



    A word of warning, without a background color and a border it is quite difficult to notice the padding in action. You just will see your piece of text moving up and down or from side to side, almost as if you were adjusting the margin instead.

     Border: The border is the limit between the padding and the margin. It can be made explicit to enhance an element by styling the border in a certain way, changing its color, thickness and style (solid, dotted, dashed, double, groove, ridge, inset, outset, none or hidden).

     Margin: The margin property defines the outermost portion (area) of the box model, creating space around an element, outside of any defined borders. The margin property is used to separate an element from its neighbor elements. The size of the margin area is determined by the margin-top, margin-right, margin-bottom, margin-left, and shorthand margin properties. Margins are set using lengths, percentages, or the keyword auto and can have negative values.

A good way to think about these areas and how they interact with each other, is to imagine you are decorating a wall (your page) with photographs (your content). You could place a mat (padding) behind a photo, you can frame (border) another photo, and you can put them in different locations (margin) relating to each other and the wall.
